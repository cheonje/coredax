코어닥스에서는 아래와 같은 API 서비스를 제공합니다.
HTTPS 프로토콜을 사용한 https://api.coredax.com/을 통해 API 접속이 가능합니다.
API는 마이페이지의 API 설정에서 발급가능합니다.

※ Public API
  • 1초당 20회 요청 가능합니다.
  • 20회 초과 요청을 보내면 API 사용이 제한되며, 제한 상태 해제는 관리자 승인이 필요합니다. (1:1문의 또는 고객센터)
※ Private API
  • 1초당 10회 요청 가능합니다.
  • 10회 초과 요청을 보내면 5분간 API 사용이 제한됩니다.
